Background music for AutoBE
===========================

Place one or more audio files here (.ogg, .mp3, or .wav) to use as optional
background music in AutoBE. Enable "Play background music" in Settings.

If no file was included with the installer, add your own royalty-free track
(e.g. convert from the credited source in MUSIC_CREDITS.txt in the AutoBE
folder). Name the file anything you like (e.g. background.ogg).
